# 🐍 Snake Game v1.0

## 🎮 How to Play

- **Desktop**: Arrow keys (↑↓←→) or WASD to control the snake
- **Mobile**: Swipe on the game canvas to change direction

## 📋 Game Rules

1. Eat red food to score **+10 points** and grow longer
2. Hitting walls or your own body → Game Over
3. **Wall Wrap Mode**: When enabled, the snake passes through walls
4. **Power-ups**: Yellow = Speed Boost (double score), Blue = Slow Down
5. Movement speed increases every **100 points**
6. **Shrink Bonus Fruit** (orange-red, shrinking): Dynamic score up to 80 points — the faster you eat, the higher the score
7. **Stealth Egg** (rare white glow): Grants temporary invincibility — ignore walls and self-collision

## 🎨 Custom Skins

Built-in backpack system for snake skins, backgrounds, and food styles. Click the HUD panel on the left to cycle through options — settings are saved automatically.
